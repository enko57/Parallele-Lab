# Parallele-Lab Project

Ein neues Projekt basierend auf einer modernen Microservices-Architektur mit Docker, separatem Frontend und Backend-Services.

## Architektur-Ideen
- **Frontend**: React (statt Angular)
- **Backend-Services**: Authentifizierung, Kern-API (z.B. für ein Aufgabenmanagement oder Buchungssystem)
- **Datenbank**: PostgreSQL
- **Deployment**: Docker Compose
